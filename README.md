#  Student Productivity Dashboard

A simple and clean React-based productivity app built as my first project 

## ✨ FEATURES

*  Add and manage tasks
*  Notes section (coming soon)
*  Pomodoro timer
*  Simple and clean UI

##  LANGUAGES

* React
* JavaScript
* HTML & CSS

##  Getting Started

Clone the repo and run locally:

```
npm install
npm start
```
